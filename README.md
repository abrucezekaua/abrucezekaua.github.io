Política de Privacidade – ProofAds

A ProofAds respeita a privacidade dos usuários e está comprometida com a proteção de seus dados.

Dados coletados:
Coletamos informações básicas de conta (e-mail) e dados de campanhas de anúncios autorizados pelo usuário por meio da Meta Ads API.

Uso dos dados:
Os dados são utilizados exclusivamente para gerar relatórios simples de desempenho de anúncios, conforme solicitado pelo usuário.

Compartilhamento:
Não compartilhamos dados com terceiros.

Armazenamento e segurança:
Os dados são armazenados de forma segura e protegidos contra acessos não autorizados.

Exclusão de dados:
O usuário pode solicitar a exclusão de seus dados a qualquer momento, conforme descrito na seção “Exclusão de dados do usuário”.
